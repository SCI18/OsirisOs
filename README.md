# OSIRIS-OS. BUILT BY DEVS, FOR DEVS!
Long term project about systems development and understsanding the inner workings of an os.
This custom built os is made to be entirely modular (for the x86/x64 systems including a termux proot environment) but also stable and safe for the daily users hence the 3-way build philosophy.

    A) Osiris Ramsees - stable mobile port. not for tinkerers
    B) Osiris Netjeru - the "tinkerers' " choice. every process 
       available to the user, cli tools, ide, etc.
    C) Osiris OS - the workstation system. a direct comparision would
       be how debian sits on a laptop/pc

The osiris-rm.json serves as not only a roadmap but a documentation of the build ( build env, tools, language, components, etc ) in full overview. The smaller, more specific details can be found as a comment on the repo at every update with versions showing the build sequence. 

This is done entirely as a learning project however, should the system actually work and go live, everything done upto that point is fully open-source and available to anyone with access to github. Any contributions to the build are welcome, including issues, and will be taken seriously because

        "Osiris OS is built by devs, for devs."
